# How to generate the apk to Android

- Windows: 
  #### $ cd .\android\
  #### $ ./gradlew assembleRelease for release 
  #### $ ./gradlew assembleDebug for debug
  Will generate a apk file in:
  ####  ..\android\app\build\outputs\apk\release 
  OR
  ####  ..\android\app\build\outputs\apk\debug
